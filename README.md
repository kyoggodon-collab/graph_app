 概要
Excelなどからコピーしたデータを利用して、複数軸（Y1〜Y7など）の折れ線グラフを直感的に作成・微調整できるデスクトップツールです。
「データはあるけど、標準のグラフ機能だと多軸がきれいに作れない…」という悩みを解決するために作成されました。

 主な機能
一瞬で読み込み: コピーしたデータを、クリップボード経由（Ctrl+V）で直接インポートできます。

直感的なズーム: Ctrl + マウスホイール で、画面上のグラフ表示を拡大・縮小できます。

高度なグラフ調整: プロット領域の縦横幅や、複数軸の間隔（オフセット）をスライダーで細かく調整可能です。

レポート向け凡例設定: 凡例の名前変更、背景の透過、枠線の非表示切り替えに対応しています。

自動単位変換: [V] のデータで数値が小さい場合、自動的に [mV] に変換して見やすくスケーリングします。

 使い方
データの準備: 表計算ソフトやテキストエディタでデータをコピーします（見出しも含めてください）。

ツールの起動: アプリケーション（Windowsなら .exe、Macなら .app）をダブルクリックして起動します。

データの貼り付け: ツール右上の「クリップボードから読込」ボタンを押すか、画面上で Ctrl+V を押します。

グラフの調整: 右側の【操作パネル】から、プロット幅や軸の間隔、凡例などを調整します。

保存: グラフ上部のフロッピーディスクのアイコン（保存ボタン）を押して、画像ファイルとして保存します。

📊 データの形式について
以下のように、1列目が「X軸」、2列目以降が「Y軸」となるようなデータに対応しています。1行目には必ずデータの名前（見出し）を含めてください。

【データの入力例】
温度 [°C]  出力電圧 Vout [V]  電流 ic [mA]  周波数 f [MHz]
-20    0.0145      1.53239    3.0012439
-10    0.0145      1.54010    3.0012483
0     0.0151      1.54870    3.0012593

⚠️ 注意事項・トラブルシューティング
初回起動時の警告（Windows）:
セキュリティ機能により、初回起動時に「WindowsによってPCが保護されました」という青い画面が出ることがあります。その場合は「詳細情報」をクリックし、「実行」ボタンを押して進めてください。

初回起動時の警告（Mac）:
「開発元を検証できません」という警告が出た場合は、ファイルを「右クリック（または二本指タップ）」して「開く」を選択してください。

起動速度について:
プログラムの構造上、初回起動時のみ立ち上がるまでに数秒〜十数秒かかる場合があります。何も起きないように見えても、裏側で準備をしていますので少しだけお待ちください。
